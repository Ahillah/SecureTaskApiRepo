
# Secure Task API

# Overview
Secure Task API is a backend application designed for managing tasks with user authentication and request logging. The project is structured following **Onion Architecture**, ensuring a clean separation of concerns between the Domain, Repository, Service, and Presentation layers.

# Features
- JWT-based user authentication and authorization.
- CRUD operations for tasks with proper validation.
- Logging of all HTTP requests and responses, including status codes, response time, and user information.
- Each log entry is associated with a user.
- Proper handling of nulls and validation errors in services.
- Clean layering and maintainable code structure.
